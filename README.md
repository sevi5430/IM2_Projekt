# WHO AM I? – Premier League Ratespiel

Ein browserbasiertes Ratespiel rund um die Premier League. Ziel ist es, einen zufällig ausgewählten Spieler der aktuellen Saison 2024/25 innerhalb von 10 Versuchen zu erraten.

## Spielprinzip

Nach jeder Eingabe eines Spielernamens wird angezeigt, wie nah der Tipp am gesuchten Spieler liegt. Die fünf Kategorien werden grün (richtig) oder rot (falsch) markiert:

| Kategorie | Beschreibung                           |
| --------- | -------------------------------------- |
| Nation    | Nationalität des Spielers              |
| Verein    | Aktueller Klub                         |
| Alter     | Lebensalter                            |
| Position  | Torwart, Abwehr, Mittelfeld oder Sturm |
| Liga      | Immer Premier League                   |

## Technologien

- HTML, CSS, JavaScript
- [football-data.org API](https://www.football-data.org/) – Spielerdaten der Premier League
- PHP als serverseitiger API-Proxy
