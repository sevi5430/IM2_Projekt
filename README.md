# WHO AM I? – Premier League Ratespiel

Ein browserbasiertes Ratespiel rund um die Premier League. Ziel ist es, einen zufällig ausgewählten Spieler der Saison 2024/25 innerhalb von 10 Versuchen zu erraten.

## Spielprinzip

Ziel des Spiels ist es, innerhalb von zehn Versuchen den gesuchten Spieler zu erraten. Als Hilfestellung wird dir nach jedem Rateversuch in fünf Kategorien angezeigt, wie nah du am gesuchten Spieler bist. Die Kategorien sind Nationalität, Verein, Position, Alter und Liga.

Beispiel: Du tippst auf Erling Haaland. Die Nationalität leuchtet rot – der gesuchte Spieler kommt also aus einem anderen Land. Die Position leuchtet grün – beide Spieler teilen dieselbe Position. Der Verein leuchtet rot – der gesuchte Spieler spielt nicht bei Manchester City. Das Alter leuchtet rot – der Spieler ist anders alt als Erling Haaland. Mit diesen Infos kannst du deinen nächsten Tipp gezielt einschränken.

| Kategorie | Beschreibung                              |
| --------- | ----------------------------------------- |
| Nation    | Nationalität des Spielers                 |
| Verein    | Verein des Spielers in der Saison 2024/25 |
| Alter     | Lebensalter in der Saison 2024/25         |
| Position  | Torwart, Abwehr, Mittelfeld oder Sturm    |
| Liga      | Immer Premier League                      |

## Technologien

- HTML, CSS, JavaScript
- [football-data.org API](https://www.football-data.org/) – Spielerdaten der Premier League Saison 2024/25
- PHP als serverseitiger API-Proxy
